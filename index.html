What is HTML?

HTML (HyperText Markup Language) is the standard language for creating webpages. It provides the structure of the website like headings, paragraphs, images, and links.

What is CSS?

CSS (Cascading Style Sheets) is used to style the webpage. It controls colors, layouts, fonts, and spacing to make pages look attractive.

What is Git?

Git is a version control system. It helps developers track changes, collaborate, and manage code effectively. Popular platform: GitHub.
